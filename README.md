# Metro FastAPI

Приложение на `FastAPI`, которое ищет кратчайший путь по московскому метро.


## Структура проекта

- `main.py` - точка входа и запуск `uvicorn`;
- `api.py` - FastAPI и HTTP-эндпоинты;
- `metro_service.py` - построение графа метро и алгоритм Дейкстры;
- `settings.py` - константы и пути к файлам;
- `templates/index.html` - страница интерфейса;
- `static/app.js` и `static/styles.css` - логика и стили фронта.

## Как запустить

1. Создайте и активируйте виртуальное окружение.
2. Установите зависимости:

```
pip install -r requirements.txt
```

3. Запустите сервер:

```
uvicorn main:app --reload

или

python main.py
```

После запуска документация будет доступна по адресам:

- `http://127.0.0.1:8000/ui`
- `http://127.0.0.1:8000/docs`
